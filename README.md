# FitMe
[Live FitMe](https://fitme-front-end.herokuapp.com/)
#### The application give you selection of exercises with dumbbells for different muscle groups. This set of exercises is good for fullbody workout at home. 
#### Functions: Exercises with description and animation Special workout routines. Full body workouts and workouts for different groups of muscles - legs, back, arms, shoulders, chest Tracking your own workout for the day

### Technologies used in the application
##### Flask, Python, React, SQLite, Tailwindcss.
